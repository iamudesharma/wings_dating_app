# Product Overview

Wings Dating App is a cross-platform Flutter dating application that enables users to discover people, build rich profiles, chat in real-time, and receive AI-assisted guidance.

## Core Features

- **Authentication**: Firebase UI with Email, Phone, and Google OAuth
- **Profile Management**: Rich profiles with photos, personal details, interests
- **User Discovery**: Paginated browsing with distance awareness and advanced filtering
- **Real-time Chat**: 1:1 messaging powered by Stream Chat
- **Engagement Tracking**: Profile visits, taps, and favorites
- **AI Wingman**: AI chat assistant and profile analysis
- **Multi-platform**: iOS, Android, Web, macOS, Windows, Linux support

## Target Users

People looking for meaningful connections through a modern, AI-enhanced dating platform with comprehensive filtering and real-time communication capabilities.

## Key Differentiators

- AI wingman for dating guidance and profile optimization
- Cross-platform availability (mobile, web, desktop)
- Advanced filtering and discovery features
- Real-time presence and engagement tracking
